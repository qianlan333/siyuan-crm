Agent 接入指南

本文档面向开发者和 AI Agent（如 Claude Code）。Agent 可直接阅读本页面并调用所有 API。

Base URL

https://<your-domain> — 所有路径均相对于此域名。

认证方式

  • 登录态 大部分接口使用 Session Cookie 认证。先通过 GET /login 或 POST /login 获取 session，后续请求携带 Cookie 即可。
  • 公开 问卷前台提交、企微回调等不需要认证。
  • 写操作需在请求参数中传 admin_action_token（从 session 获取）或 confirm=true 防止误触发。

请求约定

  • 请求体使用 Content-Type: application/json
  • 所有响应格式统一：{"ok": true, ...payload} 或 {"ok": false, "error": "..."}
  • 分页使用 limit + cursor 或 limit + offset
  • 路径参数用 <param> 标记，如 /api/customers/<external_userid>

Agent 调用建议

  • 先查看下方「API 快速索引」定位所需接口
  • 点击展开接口卡片查看参数和 curl 示例
  • 每个接口的 curl 示例可直接复制执行

把文档发给 AI

  • 每个接口卡片右侧的「复制 MD」按钮 — 复制单个接口的 Markdown 给 AI
  • 每个分组标题旁的「复制此分组」按钮 — 复制整组接口
  • 下方「复制全部 API 文档」按钮 — 复制全部 {{ endpoint_count }} 个接口的完整 Markdown，粘贴到 AI 对话即可
📋 复制全部 API 文档 — 一键复制所有接口的 Markdown 格式文档（约 {{ markdown_size_label }}），粘贴给 Claude / ChatGPT 等 AI 即可调用。
