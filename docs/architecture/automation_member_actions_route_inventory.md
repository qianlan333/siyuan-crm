# Automation Member Actions Route Inventory

Legacy Exit group 23 locks automation member detail and member actions to Next read-model / CommandBus safe-mode routes. The production_compat exact member route and member wildcard rollback are removed. This group does not take over stage manual-send, focus-send-batches, SOP, customer automation webhook / retry, real OpenClaw, real WeCom, real Bazhuayu, HTTP external calls, payment, or storage.

## Contract Notes

- If a customer detail button exists, it calls the Next exact API listed above.
- Member detail is a controlled Next read-model response; missing identity returns 400 and never falls back to the legacy facade.
- Member actions are CommandBus records plus blocked SideEffectPlan records only.
- `push-openclaw` is approval-required and plan-only; it never performs a real OpenClaw push.
- All action responses keep `route_owner=ai_crm_next`, `fallback_used=false`, `real_external_call_executed=false`, `automation_runtime_executed=false`, `openclaw_push_executed=false`, and `wecom_send_executed=false`.
