## No-Go Conditions

| condition | consequence |
| --- | --- |
| Any old production entrypoint dirty | Stop cutover until diff is isolated or reverted with backup. |
| Any fake adapter labeled as production | Stop; status is dishonest and unsafe. |
| Any parity report has blocker | Stop module promotion. |
| Real payment signing not tested | Do not enable real WeChat Pay or Alipay. |
| Real OAuth not tested | Do not enable public OAuth submit flow. |
| Real WeCom not tested | Do not enable real send/tag/media operations. |
| Database migration not reversible | Do not run production migration. |
| Frontend route missing or broken | Do not cut over that route. |
| Media Library gray smoke executes old write endpoint | Stop; media gray prep must never write old Flask. |
| Media Library real cloud or WeCom upload untested | Do not enable production media writes. |
| Product Management gray smoke executes old write endpoint | Stop; product gray prep must never write old Flask. |
