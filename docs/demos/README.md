# AI-CRM 客户侧边栏 V2 Demo 逻辑说明

## 当前确认范围

本版本只保留侧边栏内高频查看和高频操作的模块：

1. 顶部基准信息
2. 核心画像
3. 问卷
4. 商品
5. 订单
6. 素材
7. 其他客服的聊天记录

已去掉以下内容：

- AI 自动生成建议
- 建议跟进
- AI 话术推荐
- 客户分层展示
- Tab 角标数量
- 大块自动化池信息
- 工作流阶段信息，例如“第 3 天 / 重点跟进节点”
- 商品、问卷、素材里的长描述和解释性文案

## 顶部信息

顶部只展示：

- 头像
- 客户昵称
- 手机号
- 手机号已绑定状态
- 外部联系人 ID
- 更换手机号
- 当前所在工作流标题

工作流只展示标题，例如：

```text
暑期阅读转化 SOP
```

不展示阶段、不展示节点、不再使用大块“自动化池 / 自动化运营池”信息。

## 核心画像

核心画像作为默认首页，只保留 4 个字段：

1. 用户来源：可选项
2. 行业信息：可选项
3. 行业具体描述：可编辑描述题
4. 需求、卡点、跟进状态：可编辑描述题

### 用户来源选项

```text
小蓝咨询群
媛子咨询群
学员转介绍
流量群
公域
其他
公域直播流量
```

### 行业信息选项

```text
美业
大健康
知识IP
疗愈
教育培训
玄学
服装
保险
其他
餐饮
```

这里不做客户分层，也不做 AI 自动判断。

## 问卷

问卷模块只展示：

- 问卷名称
- 提交时间
- 题目数量
- 查看答案按钮

点开后才展示题目和答案。不展示问卷用途说明、不展示状态角标、不展示原始记录按钮。

## 商品

商品模块只展示：

- 商品名称
- 价格
- 发送介绍
- 查看详情

不展示商品适配说明、字段依据、适用人群描述。

## 订单

订单模块展示：

- 商品名称
- 订单号
- 金额
- 状态
- 时间
- 查看详情

不展示长说明。

## 素材

素材模块连接素材库，类型分三类：

1. 图片素材
2. 小程序素材
3. PDF 素材

所有素材类目统一展示：

- 缩略图
- 标题
- 对应标签
- 发给客户

不展示搜索框、不展示素材用途说明、不展示预览按钮。

## 其他客服的聊天记录

聊天记录模块改名为“其他客服的聊天记录”。

展示逻辑：

- 筛掉当前客服的聊天记录
- 展示筛选后的最近 20 条
- 按时间线排列
- 显示是私聊还是群聊
- 显示客户和哪位客服说的，或者在哪个群里说的
- 兼容文本和图片消息
- 不兼容其他消息类型

## GitHub Pages 使用方式

Demo 文件路径：

```text
docs/demos/customer-sidebar-v2-material-demo.html
```

如果仓库启用了 GitHub Pages，可以通过 Pages URL 直接展示。
