# 周期商品企微入口视觉增强设计

## 目标

在周期商品已付款状态页中，把“添加企微账号”从右侧小标签提升为更容易发现和点击的次级操作，同时保持底部“立即续费”为唯一主操作。

## 视觉方案

- 保留状态卡下方、详情素材上方的位置和右对齐关系。
- 按钮宽度在移动端扩展到最多 `280px`，窄屏时不超过可用内容宽度。
- 最小高度由 `30px` 提升到 `44px`，字号由 `12px` 提升到 `16px`。
- 保留圆角描边形态，使用更清晰的蓝色描边和浅蓝背景，不改成实心主按钮。
- 保留按下反馈、键盘焦点、二维码弹窗和资格显隐逻辑。

## 边界

- Capability owner 仍为 `service_period`。
- 不修改 `/s/{link_slug}` 的数据流、OAuth、权益判断、渠道码解析或底部续费 CTA。
- 不新增路由、数据库变更、外部调用或生产数据写入。
- 回滚方式为撤回本次 CSS 和对应视觉契约测试。

## 验收

- 390px 左右移动端视口中，按钮达到参考红框的尺寸层级并保持右对齐。
- 文案单行完整显示，触控高度不低于 `44px`。
- 未付款、过期或无渠道码时仍隐藏。
- 点击仍打开原有渠道码弹窗，底部续费按钮不受影响。
